# FlowResult Website

Moderne one-pager website voor FlowResult - Persoonlijke outreach die écht werkt.

## Over FlowResult

FlowResult helpt bedrijven om sneller en persoonlijker in contact te komen met hun ideale klanten door gepersonaliseerde outreachcampagnes via e-mail en LinkedIn.

## Technologie

- Pure HTML5
- CSS3 (Custom design systeem)
- Vanilla JavaScript
- Responsive design
- Geen externe frameworks

## Structuur

```
Website/
├── index.html          # Hoofdpagina (one-pager)
├── styles.css          # Styling
├── scripts.js          # Interacties en animaties
└── README.md          # Deze file
```

## Lokaal Draaien

1. Clone deze repository
2. Open `index.html` in je browser
3. Klaar!

Geen build proces of installatie nodig.

## Deployment

Deze website kan gehost worden op:
- GitHub Pages
- Netlify
- Vercel
- Elk statisch hosting platform
